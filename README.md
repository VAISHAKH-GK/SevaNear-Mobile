# SevaNear - Mobile

This is repository contains source code for the SevaNear mobile application. SevaNear that helps patients and bystanders discover free services near hospitals in Kozhikode.

## Key Features

- Free food
- Free medicines
- Dialysis
- Chemo
- Room & Shelters

## Service Details 

Service will include following details.

- Hospital
- Type of Service
- Provider
- Description
- Timings
- Eligibility 
- Documentes required
- Location
- Contact

## Impact

- Helps patients save money
- Reduces stress for families
- Ensures free services reach the right people
- Encourages community participation & transparency
